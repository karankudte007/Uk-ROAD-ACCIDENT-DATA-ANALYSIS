# Uk-ROAD-ACCIDENT-DATA-ANALYSIS

UK Road Accident Dashboard

Project Overview
The **UK Road Accident Dashboard** is an interactive data visualization project designed to analyze road accidents, casualties, and contributing factors across different dimensions such as time, vehicle type, road type, and districts.

This dashboard helps uncover patterns and insights that can support road safety improvements and data-driven decision-making.

Objectives
* Analyze accident trends over time
* Identify high-risk vehicle types and road types
* Compare casualties across districts
* Understand the relationship between number of vehicles and casualties

Key Metrics
* **Total Casualties:** 25K
* **Average Speed:** 31.09

Dashboard Insights
Accident Trends by Quarter
* Accidents increase from **Q1 to Q2**
* Slight dip in **Q3**
* Peak observed in **Q4**
* Indicates possible seasonal impact on road accidents

Vehicle Type Analysis
* **Cars** contribute to the highest number of accidents (~16K)
* Followed by motorcycles, vans, and goods vehicles
* Suggests cars are the major contributors to road incidents

Road Type Analysis
* **Single carriageways** account for ~81% of accidents
* Other road types contribute significantly less
* Highlights high-risk road infrastructure

District-wise Casualties
* **Westminster** has the highest casualties
* Followed by Lambeth and Barnet
* Useful for identifying high-risk regions

Vehicles vs Casualties
* Scatter plot shows a relationship between number of vehicles and casualties
* Higher vehicle count tends to increase accident impact

Filters Available
* **District Filter** – Analyze specific regions
* **Accident Severity Filter** – Focus on severity levels

 Key Insights
* Cars are the most involved vehicle type in accidents
* Single carriage roads are the most accident-prone
* Accident frequency is highest in Q4
* Certain districts consistently report higher casualties
* Increase in vehicles can lead to higher casualty rates
